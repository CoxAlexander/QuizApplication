Installation process
1)
After downloading all the files from the github unzip the folder and make sure that they all stay together.
2) 
Make sure that tkinter, pillow, sqlite3, and itertools are all downloaded onto the machine to run the code.
3) 
After the libraries are installed running the application using an ide that can accept python files will run the code
4) 
if anything in the main file seems to be working incorrectly contact the help desk or attempt to redownload the files again
